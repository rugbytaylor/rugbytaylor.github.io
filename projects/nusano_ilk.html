Industrial Interlock Hardware

Objective

Design an analog safety interlock system that monitors the current flowing through a machine's power wire using an open-loop Hall-effect current sensor. If the measured current stayed above a set threshold, the interlock would close and allow normal machine operation. If the current dropped below that threshold, the interlock would open, cutting off operation and allowing safe maintenance access. This was a solo project completed over about one month, spanning four design iterations from initial concept to validated hardware.

Key Requirements

  • Primarily analog circuitry, since this is a safety-critical device
  • Adjustable current trip threshold
  • Industrial environment operation
  • No specific formal safety rating targeted, but designed and treated as a safety device

System Architecture

An open-loop Hall-effect sensor measures the current on the monitored wire and outputs a proportional analog signal. That signal feeds into a comparator, which compares it against an adjustable reference voltage (Vref) set by an onboard potentiometer. The comparator's output drives an ILK safety relay, closing it above the current threshold and opening it below the threshold. The board also includes a set of indicator LEDs for quick visual status of the interlock state.

In parallel, the current sensor's output also feeds into an op amp scaling circuit to boost resolution before the signal reaches the PLC's ADC, giving the PLC a much finer-grained reading of the current than the raw sensor output alone would allow. The board also controls an external solid-state relay and interfaces with the PLC for continuous data sampling.

Design Challenges and Iterations

Getting a wide, usable adjustment range on the Vref potentiometer proved to be the biggest challenge. The first version could only tune the trip threshold across a narrow 0.5-1.5A window. After reworking the reference divider and comparator biasing across several iterations, the final version tunes smoothly across 0-3A, a much wider and more practical adjustment range.

Mechanical and Enclosure Design

I designed the mechanical enclosure for this system using the same approach as the Arc Detector project, housing the PCB in an industrial-ready enclosure with clean field wiring access.

Results

The final board's adjustable Vref was verified to tune cleanly across the full 0-3A target range. Since the underlying current signal only spans 0-7.5A, this gives usable trip-point adjustment across roughly half of the sensor's total range, far better than the original design. I also validated the complete hardware loop end to end, confirming the system closes the interlock during normal operation and correctly faults open when current drops below threshold.
